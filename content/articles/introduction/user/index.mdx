SmartHR Design System は、サイトにアクセスするすべての人が、公開されているすべてのコンテンツを閲覧・参照できます。  
閲覧・参照する以外に、利用できるコンテンツもあります。
利用者によって利用方法・利用範囲が異なりますので、それぞれのページをご確認ください。
