プロダクトの画面キャプチャや、その他ウェブサイトの画面キャプチャを作成する際、一貫性があり、わかりやすく、見栄えのよい画像を作成するための簡易チェックリストです。このリストに記載されている項目を満たすと、おおよそ作成ガイドラインにあげた品質を達成できます。  
チェックする内容によっては例外があることがあります。参考リンクも参照してください。



## 内容の調整
- [ ] [氏名や日付など、実際に重複しない要素は表記を分散させている](./creation-guideline/)
- [ ] [特定の性別、年齢層、カテゴリに表記の偏りがない](./creation-guideline/)

## サンプル
- [ ] [従業員氏名は、ユーザー社の従業員に掲載されている氏名を利用している](./creation-guideline/)
- [ ] [メールアドレスは、`{名前.苗字}@example.com`の形式になっている]()
- [ ] [企業名は、`株式会社＊＊＊＊＊`になっている]()
- [ ] [従業員番号は、`0064`など半端な数字を使用している]()
- [ ] [住所、電話番号は、デザインシステムで推奨する表記になっている]()

## ウィンドウサイズ
- パソコンの場合
    - [ ] [横幅は1400pxで作成している]()
    - [ ] [高さ880px以上で作成している]()
    - [ ] [機能を象徴する重要な要素は、880px以内に収めている]()
- スマートフォンの場合
    - [ ] [横幅は375pxで作成している]()
    - [ ] [高さ760px以上で作成している]()
    - [ ] [機能を象徴する重要な要素は、760px以内に収めている]()
